**calculateCrossSelling_(currentLeads, previousLeads, calificacionIdx, deals)**

Cross-selling is identified by deals with `tipo_transaccion === 'Cross-sell'`.

Logic (explicit leadIdSet pattern):
1. Build currentLeadIds: `var currentLeadIds = {}; for (var i = 0; i < currentLeads.length; i++) { currentLeadIds[String(currentLeads[i].id_lead)] = true; }`
2. Filter deals where `currentLeadIds[String(deal.id_lead)] === true` AND `deal.tipo_transaccion === 'Cross-sell'`
3. Count these cross-sell deals. For segmentation, look up the LEAD's segment: for each qualifying deal, get `calificacionIdx[String(deal.id_lead)]` then call `getSegment_` logic to determine manufacturers/individuals.
4. Repeat for previousLeads: build `previousLeadIds` set, filter deals similarly.
5. Use buildSegmentedMetricWithDelta_ pattern. The metricFn receives a leads subset; inside it:
   a. Build `var leadIds = {}; for (var i = 0; i < leadsSubset.length; i++) { leadIds[String(leadsSubset[i].id_lead)] = true; }`
   b. Filter deals where `leadIds[String(deal.id_lead)] === true` AND `deal.tipo_transaccion === 'Cross-sell'`
   c. Return the count of matching deals

Return format:
```javascript
{
  crossSellDeals: { total: {count, pct, delta}, manufacturers: {...}, individuals: {...} }
}
```

Note: `deals` must be passed as a parameter since it is loaded in getSDRReport. Update the stub function signature AND the call site in getSDRReport to pass `deals`.

**calculateSemaforoContesto_(currentLeads, previousLeads, calificacionIdx, interaccionesIdx)**

For each channel+toque combination, count leads that have at least 1 interaction matching that channel, toque number, and resultado = 'Contesto'.

Channels and toques (from requirement ANLYT-09):
- Telefono: toques 1, 2, 3
- WhatsApp: toques 1, 2, 3
- Correo: toques 1, 2, 3, 4

For each combination, create a metricFn that:
1. Takes a leads subset
2. For each lead, look up interaccionesIdx[lead.id_lead]
3. Check if any interaction matches: tipo_interaccion === channel AND numero_toque === toque AND resultado === 'Contesto'
4. Count leads that match

Use buildSegmentedMetricWithDelta_ for each cell.

Return format -- a nested object:

IMPORTANT: The `pct` for semaforo should be relative to the total leads in that period (what percentage of leads got a 'Contesto' on Telefono toque 1, etc.).
