**1. Semaforo Contesto (semaforoContesto section)** — Grid matrix layout per locked decision:

The semaforo table is DIFFERENT from the standard segment table. Instead of rows=metrics, it's a 2D grid where:
- Rows = channels (Telefono, WhatsApp, Correo)
- Columns = toque numbers (1, 2, 3, 4 for Correo)
- Each cell shows Count for Total segment ONLY (the grid is already complex enough)
- Include a small delta indicator per cell

However, per CONTEXT.md, the semaforo should still show segment breakdown. Since showing full Count|%|Delta x 3 segments per grid cell would be too dense, render it as: each cell shows Total count + delta. Below the grid, add a note "Desglose por segmento disponible en datos completos."

ALTERNATIVE (better approach per CONTEXT.md): Render the semaforo as ONE grid per segment. Use tabs or a segment selector within the section to toggle between Total / Manufacturers / Individuals views of the grid.

SIMPLEST approach (recommended for v1): Render semaforo with a segment selector dropdown within the section. Default to "Total". The grid shows count + delta for the selected segment.

```html
<!-- Inside semaforoContesto section-body -->
<div style="margin-bottom: 8px;">
  <select v-model="semaforoSegment" class="filter-select" style="font-size:0.78rem;">
    <option value="total">Total</option>
    <option value="manufacturers">Manufacturers</option>
    <option value="individuals">Individuals</option>
  </select>
</div>
<div class="report-table-wrapper">
  <table class="report-table semaforo-grid">
    <thead>
      <tr>
        <th>Canal</th>
        <th v-for="n in maxToquesContesto" :key="n">Toque {{ n }}</th>
      </tr>
    </thead>
    <tbody>
      <tr v-for="row in semaforoContestoGrid" :key="row.channel">
        <td class="col-metric">{{ row.channel }}</td>
        <td v-for="(toque, idx) in row.toques" :key="idx" style="text-align:center;">
          <template v-if="toque">
            <div style="font-weight:600;">{{ getSegmentValue(toque, semaforoSegment).count }}</div>
            <div style="font-size:0.7rem;">
              <span v-if="Number(getSegmentValue(toque, semaforoSegment).delta) > 0" class="delta-positive" style="font-size:0.7rem;">
                <span class="material-symbols-outlined" style="font-size:0.7rem;">arrow_upward</span>{{ getSegmentValue(toque, semaforoSegment).delta }}%
              </span>
              <span v-else-if="Number(getSegmentValue(toque, semaforoSegment).delta) < 0" class="delta-negative" style="font-size:0.7rem;">
                <span class="material-symbols-outlined" style="font-size:0.7rem;">arrow_downward</span>{{ getSegmentValue(toque, semaforoSegment).delta }}%
              </span>
              <span v-else class="delta-neutral" style="font-size:0.7rem;">
                <span class="material-symbols-outlined" style="font-size:0.7rem;">remove</span>0%
              </span>
            </div>
          </template>
          <span v-else style="color:var(--text-muted);">--</span>
        </td>
        <!-- Fill empty cells for channels with fewer toques -->
        <td v-for="n in (maxToquesContesto - row.toques.length)" :key="'empty-' + n" style="text-align:center; color:var(--text-muted);">--</td>
      </tr>
    </tbody>
  </table>
</div>
```
