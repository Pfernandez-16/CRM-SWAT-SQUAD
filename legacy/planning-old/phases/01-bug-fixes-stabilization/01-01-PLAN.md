```javascript
const isAE = userRole.value === 'AE';
```

```javascript
const isDealSource = (selectedLead.value._source === 'deal' || selectedLead.value._source === 'cross_deal');

```

This ensures after saving a cross_lead, the local leads array (not deals) gets updated, so the UI correctly reflects the change.

IMPORTANT: Do NOT change anything else in saveLeadChanges(). The EDITABLE_KEYS array, the updates collection loop, the success/failure handlers (except the dataset line), and the UI update logic (lines 516-533) remain unchanged.
